deleted_entity¶

The Deleted Entity Model.

PostgreSQL Definition¶

The deleted_entity table is defined in the MusicBrainz Server as:

CREATE TABLE deleted_entity (
    gid UUID NOT NULL, -- PK
    data JSONB NOT NULL,
    deleted_at timestamptz NOT NULL DEFAULT now()
);
